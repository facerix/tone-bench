About TONEBENCH

TONEBENCH is a procedural SFX synthesis tool built using standard web technologies by Rylee Corradini. All synthesis runs locally in your browser via the Web Audio API — no samples, no audio libraries. Shape a sound's oscillator/noise source, ADSR envelope, filter, distortion, delay, and reverb, preview it live, then export it as a WAV file or a portable code snippet. It's a Progressive Web App, meaning you can install it directly to your tablet/phone/desktop like a regular app, and your saved sound sets are stored on your device.
